Welcome to ANZ MetLite v2 Metadata Editor

The primary purpose of ANZmetLite v2 is to provide a simplified tool for creating ISO 19115-3 complete metadata for spatial resource that adheres to the Best Practice guidance provided by the ICSM Metadata Working Group.

ANZMetLite v2 currently provides templates for the following:

ANZMetLite Template
Generic ISO19115-3 following the advice of ISCM Metadata Best Practices for Datasets and Services
Emergency Management Template
Modified ANZMetLite template to suit ENSIMA requirements for metadata for spatial Datasets
ACS Template
Modified ANZMetLite template to suit Australian Climate Services requirements for metadata for spatial Datasets and Services
(Beta) GNSS Metadata Profile support
This ANZ MetLite v2 editor also provides support for the beta version of the ISO19115-3 GNSS metadata profile as developed by Geoscience Australia and Curtin University.
